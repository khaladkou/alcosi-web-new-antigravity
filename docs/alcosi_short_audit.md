## 1) Функционал сайта (коротко)

- Публичный сайт‑витрина IT/Software компании *Alcosi Group*: презентация услуг, кейсов и новостей.
- Основные разделы навигации: **Services** (AI, Fintech), **Portfolio**, **Blog**, **Contact**.
- Мультиязычность: переключатель языков **EN / PL / EE**.
- Blog/News: список статей + фильтры по категориям (Artificial intelligence/AI, Fintech, Other), отдельные страницы категорий, страницы постов.
- Portfolio: кейсы по направлениям **AI / Fintech / Blockchain** (есть отдельные страницы кейсов).
- Contact: контактный раздел (отдельный маршрут) + контактные данные (встречаются и в подвале блога).
- Документы: **Privacy Policy** опубликована отдельным PDF.

Техническое наблюдение для переноса:
- Большая часть маршрутов отдаёт «**You need to enable JavaScript to run this app**» → похоже на **SPA‑подход (клиентский рендеринг)**.
- При этом страницы блога (список/категории/посты) доступны как HTML, а Privacy Policy — как PDF.
